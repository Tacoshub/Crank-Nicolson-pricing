std::vector< double > Tridiag::solve ( std::vector< double > b)

Solves the system of equations for the tridiagonal matrix.

Solves the system of linear equations ( A \cdot x = b ), where ( A ) is the tridiagonal matrix.

Parameters
bRight-hand side vector.
Returns
Solution vector x.

This method uses LU decomposition to split the tridiagonal matrix into a lower triangular matrix ( L ) and an upper triangular matrix ( U ). It then solves the system in two steps:

  1. Forward substitution to solve ( L \cdot y = b ).
  2. Backward substitution to solve ( U \cdot x = y ).
