\doxysubsection{Detailed Description}
Represents an interest rate model with time-\/dependent rates. 

This class encapsulates the representation of an interest rate curve, providing functionality to evaluate interest rates at specific times, compute integrals over the curve, and modify rates.

The interest rate curve is stored as a vector of (time, rate) pairs, where the time points are assumed to be ordered.
