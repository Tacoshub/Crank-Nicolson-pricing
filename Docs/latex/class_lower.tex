\doxysubsection{Detailed Description}
Represents the lower triangular part of a matrix and provides a method for solving linear systems. 

The {\ttfamily \doxylink{class_lower}{Lower}} class models a lower triangular matrix (with non-\/zero entries on the diagonal and below it) and provides a method to solve systems of linear equations using forward substitution.
